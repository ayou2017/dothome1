스킵메뉴란?

키보드의 tab키를 눌렀을 때 해당 영역으로 즉시 갈수 있는 메뉴를 나타냅니다. 웹표준과 웹접근성을 지키기 위해서 반드시 만들어야합니다. 어떠한 사용자든지 어떠한 기술환경에서도 웹사이트에서 제공하는 모든 정보에 접근 할 수 있도록 보장해야합니다.

탭메뉴란?

버튼과 보여줄 컨텐츠를 연결하고 버튼을 누르면 관련된 컨텐츠가 보여집니다. 자바스크립트를 이용하여 구현할 수 있습니다.

스프라이트 효과란?

이미지 각각을 별도의 파일로 추가하는 대신, 이미지를 하나의 파일로 만들어서 포함하면 HTTP 요청 수가 줄기 때문에 메모리와 네트워크 사용량 측면에서 효율적입니다. background-position으로 컨텐츠가 보일 영역을 설정하여 이미지가 나타날 수 있게 합니다.

마우스 오버 효과란?

visibility 속성의 hidden, visible를 사용하여 사진에 마우스를 대면 "바로가기"가 나오도록 효과를 주었습니다. hidden은 콘텐츠 요소를 보이지 않게 하고, visible은 콘텐츠 요소를 보이도록 합니다.
